# PRN Viewer - Программа для просмотра и конвертации PRN файлов

## Описание

PRN Viewer - это приложение для просмотра, анализа и конвертации файлов в формате PRN. Программа предоставляет удобный интерфейс для просмотра содержимого PRN файлов, их преобразования в формат изображений (TIFF), а также создания файлов PRN.

## Основные возможности

- **Просмотр PRN файлов**: Открывайте и просматривайте содержимое PRN файлов с возможностью масштабирования и навигации
- **Конвертация в TIFF**: Сохраняйте PRN файлы в формате TIFF
- **Конвертация в PRN**: Преобразовывайте изображения в формат PRN
- **Навигация по файлам**: Перемещайтесь между файлами в текущей папке с помощью кнопок "Предыдущий" и "Следующий"
- **Информация о файле**: Отображение подробной информации о файле, включая размеры, DPI, глубину цвета и тип цвета
- **Поддержка больших файлов**: Эффективная обработка больших PRN файлов

## Особенности

- **Высокая производительность**: Быстрая загрузка и отображение даже больших файлов
- **Поддержка различных форматов**: Работа с различными форматами изображений (BMP, PNG, JPEG, TIFF и др.)
- **Информация о метаданных**: Отображение информации о DPI, цветности и других параметрах изображения
- **Удобный интерфейс**: Интуитивно понятный интерфейс с кнопками масштабирования и навигации
- **Поддержка отмены операций**: Возможность отмены длительных операций загрузки или сохранения

## Технические характеристики

- **Платформа**: .NET 9.0 Windows Forms
- **Архитектура**: x64
- **Совместимость**: Windows 10 и выше
- **Тип приложения**: Desktop GUI приложение
- **Размер**: Маленький размер исполняемого файла благодаря оптимизации

## Установка

1. Убедитесь, что на вашем компьютере установлен .NET 9.0 Runtime (если нет, то при первом запуске будет предолжено скачать с официального сайта Microsoft)
2. Скачайте и запустите исполняемый файл PRNViewer.exe

## Использование

1. Запустите приложение
2. Используйте кнопку "Открыть" для загрузки файла
3. Просматривайте изображение, используя масштабирование и навигацию
4. Сохраните в другой формат с помощью кнопки "Сохранить как TIFF" или "Сохранить как PRN"
5. Используйте кнопки навигации для перехода между файлами в папке

## Поддержка форматов

- **Входные форматы**: PRN, BMP, PNG, JPEG, TIFF и другие распространенные форматы изображений
- **Выходные форматы**: TIFF, PRN

Программа распространяеться "как есть"
